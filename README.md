## Project setup
## Backend
Change directory to `server` and import the .sql file. Then, edit `app.ini` and run
```
go run main.go
```

## Frontend
```
npm install
```

### Compiles and hot-reloads for development
Setup `.env` file, then
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
